개인정보처리방침

애플리케이션: n8n Automation

최종 업데이트: 2025년 12월 9일

요약: 이 애플리케이션은 개인 자동화 워크플로우 목적으로만 사용되며, 수집된 데이터는 제3자와 공유되지 않습니다.

1. 개요

본 애플리케이션(n8n Automation)은 개인 사용 목적의 자동화 워크플로우를 위해 Google Sheets API에 접근합니다. 본 개인정보처리방침은 애플리케이션이 어떻게 귀하의 정보를 수집, 사용, 보호하는지 설명합니다.

2. 수집하는 정보

본 애플리케이션은 다음 정보에 접근합니다:

  • Google 계정 정보: 이메일 주소, 기본 프로필 정보
  • Google Sheets 데이터: 귀하가 명시적으로 공유한 스프레드시트의 데이터

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 개인 자동화 워크플로우 실행
  • Google Sheets 데이터 읽기 및 쓰기
  • 사용자 인증 및 권한 확인

4. 정보 공유 및 제3자 제공

중요: 본 애플리케이션은 귀하의 데이터를 제3자와 공유하지 않습니다.

모든 데이터는 귀하의 개인 n8n 인스턴스 내에서만 처리되며, 외부 서버로 전송되지 않습니다.

5. 데이터 보안

귀하의 데이터 보안을 위해 다음 조치를 취합니다:

  • OAuth 2.0 프로토콜을 통한 안전한 인증
  • 암호화된 연결(HTTPS)을 통한 데이터 전송
  • 최소 권한 원칙에 따른 API 접근

6. 데이터 보관 및 삭제

귀하는 언제든지 Google 계정 설정에서 본 애플리케이션의 접근 권한을 취소할 수 있습니다. 권한 취소 시 애플리케이션은 더 이상 귀하의 데이터에 접근할 수 없습니다.

권한 취소 방법:
Google 계정 → 보안 → 타사 앱 액세스 관리 → "n8n Automation" 선택 → 액세스 권한 삭제

7. 사용자 권리

귀하는 다음 권리를 가집니다:

  • 언제든지 애플리케이션 접근 권한 취소
  • 수집된 데이터에 대한 접근 및 확인
  • 데이터 삭제 요청

8. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우 본 페이지를 통해 공지됩니다.
